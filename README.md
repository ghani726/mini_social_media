<h1>MINI Social Media MERN Stack Project</h1>


<hr/>

<h2>Tech Stack used:</h2>

<ul>
    <h3>Frontend</h3>
    <ul>
        <li>React</li>
        <li>Tailwind CSS</li>
        <li>Axios</li>
        <li>React Router</li>
    </ul>
    <h3>Backend</h3>
    <ul>
        <li>Node.js</li>
        <li>Express</li>
        <li>MongoDB</li>
        <li>ImageKit</li>
        <li>Multer</li>
    </ul>
</ul>




<h2>Features:</h2>

<ul>
    <li>Create posts</li>
    <li>View Posts</li>
</ul>


<h2>Installation:</h2>

<ol>
    <li>Clone the repository</li>
    <li>Navigate to the project directory</li>
    <li>Install dependencies for both frontend and backend</li>
    <li>Start the development server</li>
    <li>Local MongoDB server was used in it. You have to install it first or connect to a remote MongoDB cluster</li>
</ol>
